Disposable Keys

A live interactive audiovisual program that drops 3D models of piano keys in a virtual space when the piano is played.

ABOUT:

A live interactive audiovisual program that drops 3D models of piano keys in a virtual space when the piano is played.


Sound Droplets - An iteration of the dropping keys was made using extruded audio waveforms as input instead of MIDI keys.


TECHNICAL INFO:

3D audioreactive visualization is a custom program made in Touchdesigner that takes realtime MIDI input to generate the 3D geometries.
